Curated list of places where a newbie can start acquiring some bitcoin

| | | | | | | |

Originally published on Substack on Nov 29, 2022

Updated here on Dec 15, 2024

I do not want to “recommend” a specific place or exchange. New users should try to EARN directly bitcoin, instead of buying from an exchange. My personal recommendation is:

  • start accepting bitcoin with your business
  • start selling stuff for bitcoins (see section P2P Marketplaces)
  • work and be paid directly in bitcoins - see job boards section
  • or use Bitwage - here I wrote a dedicated guide

But I was asked a lot by new coiners from where they can start acquiring some. So I asked the Bitcoin community about which exchange they are more comfortable, then I create this list.

DISCLAIMER: I am not related with ANY of listed exchanges / companies, nor I am paid by them or any other entity. I am already paid by Bitcoin, by each of you that start using bitcoin as money. That’s all.

Here is a list of exchanges used in UK-US-EU-SA, all information was verified with many users, using these exchanges, so there's no risk, scam or issues (except the KYC part). WE STRONGLY RECOMMEND TO USE NON-KYC EXCHANGES.

If you know or have experience with other exchanges and worth add it to this list, please let me know. If you had an issue with one of these, also please let me know and explain a bit the situation so we can decide if is better to remove it from this curated list.

There's no "recommendation" or "best" exchange, the list is random so the order doesn't matter. Each exchange has his specific features and options. So choose one to start that suits your needs. Later test 1-3 more. Is better to have alternatives. Not all have the same exchange rates.

We recommend only one thing for newbies: start with a small amount to see how the process work and test your newly wallets. For your first buy you can use also those that are using vouchers or mobile apps.
