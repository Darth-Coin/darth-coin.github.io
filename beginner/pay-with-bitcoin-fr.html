Comment payer vos factures et dépenses régulières avec Bitcoin ?

| | | | | | | |

Publié à l'origine sur Substack le 04 décembre 2022

Mis à jour ici le 30 novembre 2024

La première question des nocoiners est : « Puis-je payer ma facture d'électricité avec Bitcoin ?» Je commence donc ici à établir une liste d'endroits où vous pouvez payer vos factures avec Bitcoin.

Cette liste sera mise à jour périodiquement. Si vous connaissez un service public qui accepte Bitcoin, envoyez-lui simplement le lien sur Telegram @DarthCoin ou NOSTR avec une explication simple.

Je n'ai aucune relation d'affiliation avec aucun de ces services, c'est juste une liste de services qui acceptent Bitcoin et beaucoup d'entre eux sont vérifiés auprès des utilisateurs. Aucun lien de parrainage ni programme de gain n'est impliqué.

Cette liste a pour but d'aider les utilisateurs à commencer à utiliser leurs bitcoins et à créer l'économie circulaire.

Je vous invite d'abord à écouter cet incroyable épisode de Once Bitten discutant avec les gars de Shopinbit sur l'achat de tout ce que vous voulez avec Bitcoin.

La liste est organisée en plusieurs sections :

  • Intermédiaires - ils paient des factures fiat pour vous et vous les payez en BTC
  • Fournisseurs de cartes VISA - vous débitez une carte VISA fiat avec vos BTC
  • Entreprises de services publics qui acceptent directement les BTC
  • Produits généraux - détaillants qui acceptent directement les BTC
  • Voyages, billets d'avion, locations qui acceptent directement les BTC
  • Listes générales de commerçants - listes publiques de tous les commerçants qui acceptent les BTC
  • Remboursement / récompenses - vous faites vos achats habituels et vous obtenez des sats en guise de récompense
  • Liste des marketplaces P2P utilisant Bitcoin
