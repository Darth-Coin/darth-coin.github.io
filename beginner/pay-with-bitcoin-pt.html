Como pagar suas contas e despesas regulares com Bitcoin?

| | | | | | | | |

Postado originalmente em Substack em 04 de dezembro de 2022. Atualizado em 21 de março de 2025

A primeira pergunta do nocoiner é: "posso pagar minha conta de luz com Bitcoin?" Então aqui começo a montar uma lista de lugares onde você pode pagar suas contas com Bitcoin.

Esta lista será atualizada periodicamente. Se você conhece um serviço de utilidade pública que comprovadamente aceita Bitcoin, basta enviar o link no Telegram @DarthCoin ou NOSTR com uma explicação simples.

Não tenho nenhuma relação de afiliação com nenhum desses serviços, é apenas uma lista de serviços que aceitam Bitcoin e muitos deles são verificados com usuários. Nenhum link de indicação ou programa de ganhos envolvido.

Esta lista é para ajudar os usuários a começar a usar seus bitcoins e criar a economia circular.

Vou convidá-lo primeiro a ouvir este episódio incrível de Once Bitten falando com os caras da Shopinbit sobre comprar tudo o que você quiser com Bitcoin.

A lista é organizada em várias seções:

  • Intermediários - eles pagam contas fiduciárias para você e você as paga em BTC
  • Provedores de cartões VISA - você debita um cartão VISA fiduciário com seu BTC
  • Empresas de serviços públicos que aceitam BTC diretamente
  • Produtos gerais - varejistas que aceitam BTC diretamente
  • Viagens, passagens aéreas, aluguéis que aceitam BTC diretamente
  • Listas gerais de comerciantes - listas públicas com todos os comerciantes que aceitam BTC
  • Cashback / recompensas - você faz suas compras regulares e ganha sats como recompensas
  • Lista de mercados P2P usando Bitcoin
