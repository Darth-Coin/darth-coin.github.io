Betalingen aan handelaren met Bitcoin LN in het echte leven, Bitcoin verovert de wereld

| | | | | | |

We horen altijd dat stomme excuus "maar je kunt niets betalen met Bitcoin...".

Velen weten het niet, maar Bitcoin via Lightning Network gaat overal zijn. Dus je kunt maar beter beginnen met leren hoe je het moet gebruiken. Dit zijn slechts enkele voorbeelden met video, maar er zijn er nog veel meer die niet op video zijn vastgelegd.

Stuur me meer voorbeelden en ik zal ze toevoegen aan deze lijst, via Telegram of via NOSTR.

Laten we aan de hele wereld laten zien hoe Bitcoin en Lightning werken en fiat en banken VEROUDERD maken.

Allereerst wil ik dat je alle video's bekijkt die zijn geplaatst in Paco de la India Vlog, tijdens zijn reis rond de wereld waarbij hij Bitcoin uitgaf. Geweldige ervaringen!

Heb je een bedrijf dat BTC/LN accepteert? Je kunt het promoten op BTCMap.org !

Hier heb je ook een lijst met plekken waar je je rekeningen kunt betalen met Bitcoin !
