Um guia simples listando o software disponível que você pode usar para executar um nó da Bitcoin Lightning Network e suas principais funcionalidades

| | | | | | | | |

Postado originalmente em Substack em 31 de outubro de 2021. Atualizado aqui em 13 de outubro de 2024.

Hoje é 31 de outubro de 2021 e comemoramos 13 anos desde que Satoshi Nakamoto lançou o Bitcoin Whitepaper. Para homenagear este dia especial, estou publicando este guia sobre nós Bitcoin e Lightning, para ajudar novos usuários a dar a eles mais opções para executar um nó.

OK, então agora você é um bitcoiner que começou sua jornada na toca do coelho do Bitcoin e quer se aprofundar mais na execução de um nó BTC/LN. Você já sabe como usar/gerenciar uma carteira BTC/LN, sabe como obter BTC de exchanges, sabe que não é bom deixar seu BTC em serviços de custódia, é hora de assumir a responsabilidade em suas próprias mãos. Bom…

Ou você é apenas um pequeno comerciante, tem um negócio que quer começar a aceitar Bitcoin por meio de pagamentos da Lightning Network, sem intermediários, como Bitcoin Papers declarou no Resumo:

Uma versão puramente ponto a ponto do dinheiro eletrônico permitiria que pagamentos online fossem enviados diretamente de uma parte para outra sem passar por uma instituição financeira.

Então, vamos revisar rapidamente algumas das melhores soluções que estão disponíveis agora. A ideia é ser TOTALMENTE independente e gerenciar seu dinheiro e pagamentos, sem intermediários no meio.

Tudo depende das especificidades da sua atividade empresarial, infraestrutura, orçamento, conhecimento de tecnologia que você tem, nível de "independência" que você quer ter.

Outro aspecto a mencionar: cada tipo de nó usa um código de implementação específico para Lightning Network: LND, Core Lightning, Eclair, LDK, Rust, Electrum.

Um nó Bitcoin precisa estar 24/7 online e confiável. Então esteja preparado para mantê-lo "vivo" e bem conectado à internet. Seu nó é seu próprio banco.

Antes de mergulhar na variedade de nós de desktop, podemos mencionar que também existem versões para dispositivos móveis, que são sincronização SPV, uma pequena parte dos cabeçalhos dos blocos de um servidor de nó completo confiável e estão mantendo um nó LN não custodial em seu dispositivo móvel.

Se você não está procurando executar um nó de roteamento PÚBLICO, você quer apenas para seu uso pessoal, com pequenos pagamentos, executar um nó LN PRIVADO em seu celular com esses aplicativos de carteira é mais do que suficiente:

  • Carteira Blixt (Neutrino, nó LND) - iOS, Android, macOS e em breve Linux
  • Breez (Neutrino, nó LND) - iOS, Android
  • Carteira Zeus LN (Neutrino, nó LND) - iOS, Android, macOS
  • Carteira Green LN (Greenlight, nó GDK) - iOS, Android
  • Carteira Phoenix (Electrum SPV, Nó Eclair) - iOS, Android
  • Carteira Bitkit (nó Electrum SPV LDK) - iOS, Android
  • Valet (nó Electrum SPV, Immortan) - Android
  • Electrum (nó Electrum SPV, nó Electrum) - iOS, Android
  • Bluewallet (nó Electrum SPV, LDK) - iOS, Android

Nó Umbrel - DIY e a Máquina Bitcoin

Descrição:
  • Este é um grátis, a maneira mais simples e rápida de fazer um nó DIY, com seu próprio hardware (dispositivo RPi ou caixa de PC normal/Linux) e também é totalmente grátis e código-aberto.
  • Agora também é oferecido como Máquina Bitcoin, plug'n'play, pronto para ser usado como um Servidor Pessoal, com muitos aplicativos já incluídos e blocos sincronizados.
  • No momento é um nó somente Tor, vindo com o nó Bitcoin e nó LND mais recentes, com um painel da web simplista para gerenciar seu nó Bitcoin e Lightning, configurações / status do sistema / backup / seed, Apps Store para adicionar/remover aplicativos, conectar ao menu de carteiras.
  • Depois de instalados e sincronizados os dados do blockchain, você pode comece a usá-lo como seu próprio banco, conectando muitos aplicativos de carteira e plugins de processador de pagamento aos seus aplicativos incluídos no topo do nó LND. Alguns dos aplicativos como BTCPay e LNBits, se você quiser usá-los em clearnet, exigirão algumas modificações na configuração para poderem ser "visíveis" fora da rede Tor. Guias estão incluídos.
Software usado, aplicativos disponíveis/incluídos:
  • SO Rasp DebianOS adaptado para aplicativos dockerizados (para usuários do RaspPi)
  • qualquer distribuição de SO Linux + Docker (Debian recomendado para usuários não RPi)
  • Bitcoin Core + nó LND como software de nó base + Electrs (servidor Electrum)
  • Além disso, você pode instalar e usar o nó CLN
  • AppStore: BTC RPC Ecplorer, Mempool, RTL (Ride The Lightning), Thunderhub, Lightning Terminal, BTCPay Server, LNBits, Whirlpool, Sphinx Relay, Bluewallet LNDHUB, Specter, LNMarkets, Kristall Bull, VaultWarden, NextCloud, Synapse Matrix/Element, SqueakNode, PhotoPrism, Pi-Hole, SimpleTorrent, Home Assistant, Node-RED, code-server, Gitea e muitos mais
Recursos:
  • Página oficial - Página do GitHub - Fórum da comunidade - Grupo do Telegram - Blog
  • Guia de primeiros passos com o Umbrel, explicando em detalhes como começar, como instalar e como gerenciá-lo.

myNodeBTC - DIY e Premium

Descrição:
  • myNode é um dispositivo dedicado que fornece acesso à Bitcoin e Lightning Network, juntamente com uma série de outros recursos.
  • solução de nó completa com muitas funcionalidades, você pode comprar a máquina já instalada ou pode baixar o software e construí-la você mesmo com seu próprio hardware.
  • É oferecido em 3 versões: One, Premium, Community Edition (faça você mesmo) e também você pode pedir uma máquina RPi já instalada e sincronizada (myNode One)
  • Ele pode ser executado no Tor ou clearnet conforme o usuário decidir e vem com um painel da web simplista onde o usuário pode gerenciar aplicativos e status dos serviços de nó.
  • Depois de instalado e sincronizado, o usuário pode se conectar a vários aplicativos de carteira e soluções de pagamento.
Software usado, aplicativos disponíveis / incluído:
