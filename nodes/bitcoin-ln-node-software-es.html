Una guía sencilla que enumera el software disponible que puede utilizar para ejecutar un nodo de la red Lightning de Bitcoin y sus principales funcionalidades

Hoy es 31 de octubre de 2021 y celebramos 13 años desde que Satoshi Nakamoto publicó el Libro blanco de Bitcoin. Para honrar este día especial, estoy publicando esta guía sobre Bitcoin y los nodos Lightning, para ayudar a los nuevos usuarios a tener más opciones para ejecutar un nodo.

Bien, ahora eres un bitcoiner que comienza su viaje en la madriguera profunda de Bitcoin y quieres profundizar en la gestión de un nodo BTC/LN. Ya sabes cómo usar/administrar una billetera BTC/LN, sabes cómo obtener algunos BTC de los exchanges, sabes que no es bueno dejar tus BTC en servicios de custodia, es hora de que tomes la responsabilidad en tus propias manos. Bueno…

O eres un pequeño comerciante, tienes un negocio que quiere empezar a aceptar Bitcoin a través de pagos de Lightning Network, sin intermediarios de ningún tipo, como Bitcoin Papers establece en el resumen:

Una versión puramente peer-to-peer del dinero electrónico permitiría que los pagos en línea se envíen directamente de una parte a otra sin pasar por una institución financiera.

Repasemos rápidamente algunas de las mejores soluciones que están disponibles en la actualidad. La idea es ser TOTALMENTE independiente y administrar usted mismo su dinero y pagos, sin intermediarios de por medio.

Todo depende de las particularidades de su actividad comercial, infraestructura, presupuesto, conocimientos de tecnología que tenga y el nivel de "independencia" que desee tener.

Otro aspecto a mencionar: cada tipo de nodo utiliza un código de implementación específico para Lightning Network: LND, Core Lightning, Eclair, LDK, Rust, Electrum.

Un nodo de Bitcoin tiene que estar en línea las 24 horas del día, los 7 días de la semana y ser confiable. Así que prepárate para mantenerlo "vivo" y bien conectado a Internet. Tu nodo es tu propio banco.

Antes de sumergirnos en la variedad de nodos de escritorio, podemos mencionar que también existen versiones para dispositivos móviles, que sincronizan SPV, una pequeña parte de los encabezados de los bloques de un servidor de nodo completo confiable y mantienen un nodo LN sin custodia en tu dispositivo móvil.

Si no estás buscando ejecutar un nodo de enrutamiento PÚBLICO, solo lo quieres para tu uso personal, con pagos pequeños, ejecutar un nodo LN PRIVADO en tu móvil con estas aplicaciones de billetera es más que suficiente:

  • Billetera Blixt (Neutrino, nodo LND) - iOS, Android, macOS y próximamente Linux
  • Breez (Neutrino, nodo LND) - iOS, Android
  • Billetera Zeus LN (Neutrino, nodo LND) - iOS, Android, macOS
  • Billetera Green LN (Greenlight, nodo GDK) - iOS, Android
  • Billetera Phoenix (Electrum SPV, nodo Eclair) - iOS, Android
  • Billetera Bitkit (nodo LDK Electrum SPV) - iOS, Android
  • Valet (nodo Electrum SPV, Immortan) - Android
  • Electrum (nodo Electrum SPV, Electrum) - iOS, Android
  • Bluewallet (nodo Electrum SPV, LDK) - iOS, Android

Nodo Umbrella - DIY y Bitcoin Máquina

Descripción:
  • Esta es una forma gratuita, sencilla y rápida de crear un nodo DIY, con su propio hardware (dispositivo RPi o PC/Linux normal) y también es totalmente gratuita y de código abierto.
  • Ahora también se ofrece como Máquina Bitcoin, plug'n'play, lista para usarse como un servidor personal, con muchas aplicaciones ya incluidas y bloques sincronizados.
  • En este momento es un nodo solo Tor, que viene con el último nodo Bitcoin y el nodo LND, con un panel de control web simplista para administrar su nodo Bitcoin y Lightning, configuraciones / estado del sistema / copia de seguridad / semilla, App Store para agregar / eliminar Aplicaciones, conectar al menú de billeteras.
  • Una vez que se hayan instalado y sincronizado los datos de la cadena de bloques, puedes comenzar a usarla como tu propio banco, conectando muchas aplicaciones de billetera y complementos de procesadores de pagos a sus aplicaciones incluidas en la parte superior del nodo LND. Algunas de las aplicaciones como BTCPay y LNBits, si quieres usarlas en clearnet, requerirán algunas modificaciones en la configuración para poder ser "visibles" fuera de la red Tor. Se incluyen guías.
Software utilizado, aplicaciones disponibles/incluidas:
  • OS Rasp DebianOS adaptado para aplicaciones dockerizadas (para usuarios de RaspPi)
  • Cualquier distribución de SO Linux + Docker (se recomienda Debian para usuarios que no sean de RPi)
  • Bitcoin Core + nodo LND como software de nodo base + Electrs (servidor Electrum)
  • Además, puede instalar y usar el nodo CLN
  • AppStore: BTC RPC Ecplorer, Mempool, RTL (Ride The Lightning), Thunderhub, Lightning Terminal, BTCPay Server, LNBits, Whirlpool, Sphinx Relay, Bluewallet LNDHUB, Spectre, LNMarkets, Kristall Bull, VaultWarden, NextCloud, Synapse Matrix / Element, SqueakNode, PhotoPrism, Pi-Hole, SimpleTorrent, Home Assistant, Node-RED, code-server, Gitea y muchos más
