Un guide simple répertoriant les logiciels disponibles que vous pouvez utiliser pour exécuter un nœud Bitcoin Lightning Network et leurs principales fonctionnalités

| | | | | | | |

Publié à l'origine sur Substack le 31 octobre 2021. Mis à jour ici le 13 octobre 2024.

Aujourd'hui, nous sommes le 31 octobre 2021 et nous célébrons les 13 ans depuis que Satoshi Nakamoto a publié le livre blanc Bitcoin. Pour honorer cette journée spéciale, je publie ce guide sur les nœuds Bitcoin et Lightning, pour aider les nouveaux utilisateurs à leur donner plus de choix pour exécuter un nœud.

OK, vous êtes maintenant un bitcoiner qui commence son voyage dans le terrier profond du lapin Bitcoin et qui souhaite aller plus loin dans l'exécution d'un nœud BTC/LN. Vous savez déjà comment utiliser/gérer un portefeuille BTC/LN, vous savez comment obtenir des BTC auprès des bourses, vous savez qu'il n'est pas bon de laisser vos BTC sur des services de garde, il est temps pour vous de prendre la responsabilité en main. Bien…

Ou vous êtes simplement un petit commerçant, vous avez une entreprise qui souhaite commencer à accepter les paiements Bitcoin via Lightning Network, sans aucun intermédiaire, comme l'indiquent Bitcoin Papers dans le résumé :

Une version purement peer-to-peer de la monnaie électronique permettrait d'envoyer des paiements en ligne directement d'une partie à une autre sans passer par une institution financière.

Passons rapidement en revue certaines des meilleures solutions actuellement disponibles. L’idée est d’être TOTALEMENT indépendant et de gérer soi-même son argent et ses paiements, sans intermédiaire.

Tout dépend des spécificités de votre activité commerciale, de votre infrastructure, de votre budget, de vos connaissances techniques et du niveau d’« indépendance » que vous souhaitez avoir.

Autre aspect à mentionner : chaque type de nœud utilise un code d’implémentation spécifique pour Lightning Network : LND, Core Lightning, Eclair, LDK, Rust, Electrum.

Un nœud Bitcoin doit être en ligne 24h/24 et 7j/7 et fiable. Soyez donc prêt à le maintenir « en vie » et bien connecté à Internet. Votre nœud est votre propre banque.

Avant de plonger dans la variété des nœuds de bureau, nous pouvons mentionner qu'il existe également des versions pour les appareils mobiles, qui synchronisent SPV, une petite partie des en-têtes des blocs à partir d'un serveur de nœud complet de confiance et conservent un nœud LN non dépositaire sur votre appareil mobile.

Si vous ne cherchez pas à exécuter un nœud de routage PUBLIC, que vous souhaitez uniquement pour votre usage personnel, avec de petits paiements, exécuter un nœud LN PRIVÉ sur votre mobile avec ces applications de portefeuille est plus que suffisant :

  • Blixt wallet (Neutrino, nœud LND) - iOS, Android, macOS et bientôt Linux
  • Breez (Neutrino, nœud LND) - iOS, Android
  • Portefeuille Zeus LN (Neutrino, nœud LND) - iOS, Android, macOS
  • Portefeuille Green LN (Greenlight, nœud GDK) – iOS, Android
  • Portefeuille Phoenix (Electrum SPV, nœud Eclair) - iOS, Android
  • Portefeuille Bitkit (nœud Electrum SPV LDK) - iOS, Android
  • Valet (Electrum SPV, nœud Immortan) - Android
  • Electrum (Electrum SPV, Electrum node) - iOS, Android
  • Bluewallet (Electrum SPV, LDK node) - iOS, Android

Umbrel Node - DIY et The Bitcoin Machine

Description :
  • Il s'agit d'un moyen gratuit, le plus simple et le plus rapide de créer un nœud avec votre propre matériel (appareil RPi ou boîtier PC normal/Linux) et il est également totalement gratuit et ouvert source.
  • Il est désormais également proposé sous la forme de Bitcoin Machine, plug’n’play, prêt à être utilisé comme serveur personnel, avec de nombreuses applications déjà incluses et des blocs synchronisés.
  • Il s’agit actuellement d’un nœud Tor uniquement, livré avec le dernier nœud Bitcoin et le nœud LND, avec un tableau de bord Web simpliste pour gérer votre nœud Bitcoin et Lightning, les paramètres / l’état du système / la sauvegarde / la graine, l’App Store pour ajouter / supprimer des applications, se connecter au menu des portefeuilles.
  • Une fois les données de la blockchain installées et synchronisées, vous pouvez commencer à l’utiliser comme votre propre banque, en connectant de nombreuses applications de portefeuille et des plugins de processeur de paiement à ses applications incluses au-dessus du nœud LND. Certaines des applications comme BTCPay et LNBits, si vous souhaitez les utiliser en clearnet, nécessiteront quelques modifications de configuration pour pouvoir être « visibles » en dehors du réseau Tor. Des guides sont inclus.
Logiciels utilisés, applications disponibles / incluses :
  • OS Rasp DebianOS adapté aux applications dockerisées (pour les utilisateurs de RaspPi)
  • toute distribution Linux OS + Docker (Debian recommandé pour les utilisateurs non-RPi)
  • Bitcoin Core + nœud LND comme logiciel de nœud de base + Electrs (Electrum Server)
  • Vous pouvez également installer et utiliser le nœud CLN
  • AppStore : BTC RPC Ecplorer, Mempool, RTL (Ride The Lightning), Thunderhub, Lightning Terminal, BTCPay Server, LNBits, Whirlpool, Sphinx Relay, Bluewallet LNDHUB, Spectre, LNMarkets, Kristall Bull, VaultWarden, NextCloud, Synapse Matrix / Element, SqueakNode, PhotoPrism, Pi-Hole, SimpleTorrent, Home Assistant, Node-RED, code-server, Gitea et bien d'autres plus

  • Guide de démarrage avec Umbrel, expliquant en détail comment démarrer, comment l'installer, comment le gérer.

myNodeBTC - DIY et Premium

Description :
  • myNode est un appareil dédié qui donne accès au réseau Bitcoin et Lightning ainsi qu'à un certain nombre d'autres fonctionnalités.
  • solution de nœud complète avec de nombreuses fonctionnalités, vous pouvez acheter la machine déjà installée ou vous pouvez télécharger le logiciel et le construire vous-même avec votre propre matériel.
  • Est proposé en 3 versions : One, Premium, Community Edition (DIY gratuit) et vous pouvez également commander une machine RPi déjà installée et synchronisée (myNode One)
  • Il peut fonctionner sur Tor ou clearnet selon le choix de l'utilisateur et il est livré avec un tableau de bord Web simpliste où l'utilisateur peut gérer les applications et l'état des services de nœud.
  • Une fois installé et synchronisé, l'utilisateur peut se connecter à diverses applications de portefeuille et de paiement solutions.
