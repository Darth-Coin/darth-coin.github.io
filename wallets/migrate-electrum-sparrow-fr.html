Procédure pas à pas pour importer ou migrer votre ancien portefeuille Electrum BTC vers Sparrow Wallet

| | | | | | | | |

Publié à l'origine sur Substack le 20 février 2022

Mis à jour ici le 20 décembre 2024

De nombreux utilisateurs me demandent comment utiliser la graine Electrum BIP39 dans d'autres applications de portefeuille. Alors maintenant, je vous expliquerais comment je le fais.

Alors, considérons que vous créez vos portefeuilles froids, en utilisant mes guides précédents ou que vous avez simplement une ancienne graine BIP39 et que vous souhaitez maintenant l'utiliser dans le portefeuille Sparrow.

Il existe deux méthodes pour importer un portefeuille Bitcoin précédemment créé dans Electrum, dans l'application Sparrow :

  • La méthode simple : utiliser le fichier de sauvegarde du portefeuille Electrum.
  • La méthode avancée : convertir la graine BIP39 en BIP84 avec son chemin de dérivation.

Ressources utilisées :

  • Portefeuille Electrum : page de téléchargement et page de documentation
  • Portefeuille Sparrow : page de téléchargement et page de documentation
  • Page des outils de cryptographie

A. Importer le fichier du portefeuille Electrum dans Sparrow

  1. Une fois que vous avez créé le portefeuille Electrum et que tout fonctionne correctement, allez dans le menu Fichier - Enregistrer la sauvegarde. Il vous demandera l'emplacement où l'enregistrer, cliquez sur la case et sélectionnez l'emplacement. Gardez-le en sécurité !
  2. Ce fichier est crypté ! Vous devrez donc utiliser votre mot de passe Electrum pour l'ouvrir/l'utiliser dans Sparrow. Il s'agit du mot de passe que vous avez mis (deux fois) lorsque vous avez créé le portefeuille Electrum. Ce n'est PAS la phrase de passe supplémentaire, au cas où vous en auriez utilisé une avec la graine de 12 mots. C'est le mot de passe utilisé pour ouvrir votre portefeuille Electrum.
  3. Ouvrez l'application Sparrow. Sélectionnez Importer le portefeuille et sélectionnez Electrum. Recherchez le fichier précédent enregistré au point 1. Il vous sera demandé de saisir le mot de passe du portefeuille.

Suivez simplement les étapes et Sparrow commencera à importer tous les UTXO, transactions, etc. précédents. Il vous sera demandé de définir un mot de passe pour ce portefeuille. Cela crypte également le fichier du portefeuille local et peut être enregistré ultérieurement dans un emplacement sûr (gestionnaire de mots de passe - KeePass), utilisez l'option Fichier - Exporter du menu.

Terminé !

B. Conversion avancée des graines de BIP39 à BIP84

  1. Vous avez la graine BIP39 de 12 mots et pour vérifier que vous la convertissez correctement, il est préférable d'avoir à portée de main certaines des premières adresses BTC générées, le chemin de dérivation, la MPK (clé publique principale), la clé privée de base.
  2. Accédez à https://guggero.github.io/cryptography-toolkit/#!/hd-wallet (vous pouvez enregistrer la page et l'utiliser hors ligne, pour plus de sécurité).
  3. Mettez votre graine BIP39 de 12 mots depuis Electrum

  4. Sélectionnez BIP39 et dans la clé racine du nœud HD, sélectionnez le BTC natif segwit BIP84 (ou le format que vous avez généré dans Electrum, hérité ou segwit). Dans le bloc d'écran suivant, vous verrez la conversion en BIP84

  5. Sélectionnez le type de pièce natif segwit. Enregistrez ce chemin de dérivation et tous les détails affichés dans votre gestionnaire de mots de passe. Maintenant, copiez cette sortie « Clé dérivée base58 ».
  6. Ouvrez l'application Sparrow. Sélectionnez dans le menu Fichier - Nouveau portefeuille. Choisissez un nom.

  7. Ensuite, dans l'écran suivant, vous pouvez coller la clé dérivée base58 et l'importer.

Suivez les écrans et vous verrez comment se déroule l'importation des UTXO et des transactions précédentes. Non, vous pouvez aller les vérifier, il doit s'agir des mêmes adresses BTC que celles du précédent portefeuille Electrum natif segwit (bech32).

C'est fait ! Vous avez maintenant importé votre ancien portefeuille Electrum bech32 dans Sparrow.

Mon conseil est de migrer tous vos anciens portefeuilles BIP39 HODL vers des portefeuilles de type Taproot (P2TR). Lisez ici le guide à ce sujet.
