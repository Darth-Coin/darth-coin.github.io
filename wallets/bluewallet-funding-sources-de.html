So verwenden Sie Bluewallet mit vielen verschiedenen lndhub-Kontoquellen und aktivieren den eingebetteten LN-Knoten (LDK)

| | | | | | | | |

Dieser Artikel wurde ursprünglich am 25. Februar 2023 auf Substack veröffentlicht und OP_RETURN auf Blockhöhe #778230

Aktualisiert am 07. Okt. 2024

Am 23. Feb. 2023 gab das Bluewallet-Team bekannt, dass es seinen LNDHUB-Server (lndhub.io) abschalten wird.

Ich habe gesehen, dass viele Benutzer den Text dieser Ankündigung nicht beachtet haben und ausgeflippt sind, wie die ganze Welt wird enden … Zu viel Lärm um nichts.

Was bedeutet das für normale Bluewallet (BW)-Benutzer?

  • Nicht zu viel. Ja, es betrifft direkt nur diejenigen, die ausschließlich lndhub.io als Finanzierungsquelle für ihre BW LN-Wallets verwenden.
  • Die BW-App wird NICHT ABGESCHALTET (wie viele auch ausflippten), die App kann weiterhin wie gewohnt verwendet werden, es hat nichts mit lndhub.io zu tun.
  • WENN Sie lndhub.io in Ihrer BW-App verwenden, müssen Sie die Gelder bis zum 30. April auf ein anderes LN-Wallet übertragen. Andernfalls verlieren Sie sie. Lndhub.io war nur eine Finanzierungsquelle für normale BW-Benutzer, die KEINEN eigenen Knoten mit einem lndhub-Server betreiben. Das ist alles.
  • Wenn Sie Ihre Sats nicht von diesem lndhub-Konto eingelöst haben, haben Sie immer noch die Möglichkeit, den Anweisungen in diesem Bluewallet-Beitrag zu folgen.

Welche Optionen haben Sie als Bluewallet-Benutzer?

Wir haben hier vier Kategorien:

  • diejenigen, die keinen LN-Knoten ausführen/haben (einfacher Modus)
  • diejenigen, die ihren eigenen LN-Knoten ausführen (erweiterter Modus)
  • komplett zu einem anderen LN-Wallet wechseln (Extremfall)
  • oder die neue (versteckte) Option für den LDK-Knoten verwenden in Bluewallet

A. Benutzer, die keinen eigenen LN-Knoten haben/nicht betreiben können

Zunächst einmal: Keine Panik, Sie haben noch Zeit, Ihre Gelder zu verschieben. Das ist nicht das Ende der Welt. Jetzt ist es an der Zeit, die Optionen zu analysieren und zu überlegen, ob der richtige Zeitpunkt ist, Ihren eigenen LN-Knoten zu starten oder einfach weiterhin Depotfinanzierungsquellen für Ihr LN-Wallet zu verwenden.

Wenn Sie Ihren eigenen LN-Knoten immer noch nicht betreiben können, finden Sie hier einige Alternativen, wohin Sie Ihre Gelder von diesem lndhub.io-Konto/Bluewallet-Server verschieben können.

Denken Sie daran: Dies sind immer noch Depotdienste! Wenn Sie damit einverstanden sind, können Sie Ihre Konten einrichten. Bewahren Sie nicht zu viele Gelder auf diesen Konten auf. Verwenden Sie sie nur so weit, dass Sie kleine Beträge erhalten oder kleine Zahlungen tätigen, und verschieben Sie die Gelder dann in Ihre eigenen Wallets (Onchain oder LN).

Es ist in Ordnung, Depotdienste wie diese zu verwenden, solange Sie nicht von einer verrückten KYC-Verifizierung betroffen sind oder von einem bekannten „Onkel Jim“ geleitet werden. Hier sind einige:

  • SatsMobi - Telegram LN-Bot mit umfangreichen Funktionen (LNURL, LN-Adresse, lndhub-Konto, LN-Authentifizierung, PoS, NFC-Karten, NIP-57-Nostr-Zaps und viele mehr).
  • Alby oder Alby Hub - LN-Browsererweiterung, LNURL, LN-Adresse, lndhub-Konto, LN-Authentifizierung, NIP5, NIP57 und viele weitere Funktionen.
  • LNbits - leistungsstarkes LN-Buchhaltungssystem mit umfangreichen Funktionen und Erweiterungen für viele Anwendungsfälle, endlose LNURL und LN-Adressen, LN-Auth, Konten. Kann auf Ihrem eigenen Server und LN-Knoten ausgeführt werden oder Sie können vorübergehend die Demo-Instanz verwenden.
  • BTCPay Server – leistungsstarker LN-Server speziell für Händler, die eine LN-Bank einbinden, basierend auf einem lndhub. So können Sie Ihre Bluewallet-App ganz einfach verbinden.
  • Vielleicht gibt es dort sicher noch andere, nicht öffentlich angekündigte lndhub-Anbieter. Praktisch jeder kann einen lndhub-Server betreiben und Konten für Familie und Freunde bereitstellen, wie ich in diesem Handbuch beschrieben habe.

Schritte zum Überweisen Ihrer Gelder von BW lndhub.io auf ein anderes LN-Konto

Wählen Sie einen oder alle der oben genannten lndhub-Dienste aus und eröffnen Sie ein Konto. Das ist kinderleicht. Jeder dieser Dienste bietet Anleitungen zum Erstellen Ihres lndhub-Kontos und zum Verknüpfen mit BW.

SatsMobiBot – Anwendungsfall

Gehen Sie zu Telegram und starten Sie den Chat mit @SatsMobiBot, /start, erstellen Sie das mit Ihrem Telegram-Konto verknüpfte Konto. Es ist besser, zuerst Ihren Alias ​​in Telegram einzurichten, indem Sie einen @Benutzernamen hinzufügen.

Geben Sie /link ein und Sie erhalten einen QR-Code und eine URL wie:


    lndhub://admin:[string]@https://seven.gwoq.com/lndhub/ext/

Speichern Sie diese Informationen aus Ihrem Telegram, z. B. mit einem Passwortmanager wie KeePass.

Kopieren Sie die URL oder scannen Sie den QR und öffnen Sie jetzt Ihr Bluewallet. Gehen Sie zu „Neues LN-Wallet erstellen“ und fügen Sie die URL ein oder scannen Sie den in SatsMobiBot bereitgestellten QR-Code.

Fertig! Speichern Sie das neue lndhub-Konto, geben Sie einen gewünschten Namen ein und verwenden Sie es als LN-Wallet.

Erstellen Sie nun im selben importierten SatsMobiBot-LN-Konto eine Rechnung mit einem Betrag aus Ihrem alten BW lndhub-Konto.

Nehmen wir an, Sie hatten 100.000 Sats auf diesem BW lndhub.io-Konto. Sie müssen immer 1 % als Rücklage für Gebühren berücksichtigen. Also mehrere Teile von 99 % des verbleibenden Guthabens, nicht alles auf einmal. In diesem Fall erstellen wir eine Rechnung über 99.000 Sats im neu erstellten lndhub-Konto. Kopieren Sie sie, gehen Sie zum alten BW lndhub.io, fügen Sie sie ein und bezahlen Sie sie.

Sie haben also 1.000 Sats – Gebühren. Erstellen Sie eine weitere Rechnung im neuen lndhub-Konto über 99 % des Guthabens. Und bezahlen Sie sie vom alten lndhub.io-Konto. Wiederholen Sie den Vorgang, bis Sie genug davon haben, um dem BW-Team einige Sats als Spende für seine Dienste zu hinterlassen.

Jetzt können Sie das alte LN-Konto aus Ihrer BW-App entfernen und das neue verwenden.

Denken Sie daran: Dieser SatsMobiBot bietet auch viele andere nützliche Dienste wie:

  • LNURL / Lightning-Adresse – Benutzername@sats.mobi oder anonyme LNURL/LN-Adresse
  • NIP05 – NOSTR-Verifizierung
  • NIP57 – NOSTR-Zap-Adresse
  • Shops, Sats mit Fiat kaufen, Gruppenchat-Funktionen, Faucet, Tipjar
  • Scrub – Sats automatisch an jede externe Adresse abheben
  • Eine NFC-Karte verknüpfen

Alby-Erweiterung – Anwendungsfall

Dies ist eine Browsererweiterung. Sie müssen das Konto also von einem Desktop-Computer aus erstellen. Auch hier handelt es sich um einen Depotservice, der aber auch eine Verbindung zu Ihrem eigenen Knoten bietet, wenn Sie einen betreiben.

Gehen Sie also einfach auf deren Webseite, installieren Sie die Browsererweiterung, erstellen Sie ein Konto und suchen Sie in den Einstellungen nach den Details des lndhub, die mit Ihrer BW-App verknüpft werden sollen.

Sie können auch Ihren eigenen Alby Hub verwenden, der selbst gehostet wird und mehrere Konten für Familie und Freunde hat, wie ich in diesem Handbuch beschrieben habe.

Scannen Sie diesen QR-Code mit Ihrer BW-App (erstellen Sie ein neues LN-Konto) und fertig, Sie können es wie jedes andere LN-Wallet verwenden.

Denken Sie daran, dass Sie sich mit diesem Konto mit LN-Auth anmelden und es in NOSTR als NIP-05 oder Tip Zap verwenden können Adresse oder einfach als Spendenseite wie https://getalby.com/darthcoin.

LNbits – Anwendungsfall

LNbits ist eine leistungsstarke Suite-Software, die auf einem LN-Knoten ausgeführt werden kann. Wenn Sie sie nur als vorübergehende LN-Finanzierungsquelle verwenden möchten, können Sie den öffentlichen Demo-Server https://demo.lnbits.com oder die LN Bank LN Voltz (brasilianische Community-LNbits-Bank) verwenden.

Denken Sie daran: Dies ist nur ein Demo-Depotserver! Bitte missbrauchen Sie ihn nicht! Es ist in Ordnung, Ihre Gelder für eine bestimmte Zeit dorthin zu verschieben, aber betrachten Sie es nicht als Ihre Hauptfinanzierungsquelle. Verschieben Sie Ihre Gelder später einfach von dort in Ihre eigenen LN-Wallets oder Onchain-Wallets. Sie können sogar die LNbits-Erweiterung Boltz oder Deezy verwenden, um Ihre Gelder an eine beliebige Onchain-Adresse zu transferieren, oder die Scrub-Erweiterung verwenden, um die empfangenen Gelder automatisch an einen selbstverwahrten LN-Knoten/eine selbstverwahrte LN-Adresse weiterzuleiten.

Wenn Sie die Demo oder eine andere öffentlich gehostete LNbits-Instanz verwenden, können Sie Ihr lndhub-Konto exportieren, indem Sie zur lndhub-Erweiterung gehen und die lndhub-URL scannen oder kopieren und als neues LN-Konto in Ihre Bluewallet-App einfügen.

Dasselbe können Sie auch tun, indem Sie die im vorherigen Punkt zu LightningTipBot erläuterten Schritte ausführen.

Optional können Sie Ihre eigene LNbits-Instanz ausführen, auch ohne einen LN-Knoten auszuführen, aber indem Sie eine externe Finanzierungsquelle auswählen (siehe die LNbits-Dokumentation).

Sie können beispielsweise ein kostenloses Konto bei LNpay.co oder Alby erstellen und verknüpfen Sie es als Finanzierungsquelle für Ihre selbstgehostete LNbits-Instanz. Sie können sogar den SatsMobiBot als Finanzierungsquelle für Ihre LNbits-Instanz verwenden. Das bedeutet, dass Sie die Liquidität dieser Server für Ihre eigene LNbits-Instanz verwenden! Ist ziemlich verrückt, funktioniert aber als vorübergehende Lösung erstaunlich gut. Aber die beste und einfachste Lösung ist, einen Phoenixd-Knoten als Finanzierungsquelle zu verwenden. Funktioniert sehr einfach einzurichten und erfordert kein Liquiditätsmanagement.

B. Benutzer, die ihren eigenen LN-Knoten haben

Wenn Sie in diese Kategorie fallen, haben Sie folgende Optionen:

  • Installieren Sie das BW lndhub-Repository auf Ihrem eigenen LN-Knoten. Umbrel, myNode, Citadel haben dies bereits integriert, Sie müssen es nur aus ihrem App Store installieren. Verbinden Sie sich dann mit Ihrem eigenen lndhub-Server und erstellen Sie endlose lndhub-Konten (gehostet) in Ihren Bluewallet-Apps. Hier habe ich eine Anleitung dazu geschrieben.
  • Installieren Sie die lndhub-Bibliotheken vom BW-Team Github. Stellen Sie sich und/oder Ihrer Familie und Ihren Freunden dann mehrere lndhub-Konten zur Verfügung, die in der Bluewallet-App verwendet werden können.
  • Andere Methoden, die LNbits auf Ihrem Knoten verwenden., hier ist eine Anleitung. Diese Option wird auch in meiner anderen Anleitung zum Aufbau einer LN-Bank für Ihre Community beschrieben. Dies ist eine der besten Optionen, da Sie auf diese Weise mehr Menschen in Ihrer Umgebung helfen und auch Ihren Familienmitgliedern und Freunden gute LN-Dienste anbieten, und zwar auf eine privatere Art und Weise, LN zu nutzen.
  • Verwenden Sie Lightning-Konten vom Lightning-Terminal, wenn Sie einen LND-Knoten betreiben. Hier ist die Dokumentation.

C. Wechseln Sie zu einer anderen LN-Wallet-App

Ja, das ist die einfache Möglichkeit. Aber wenn Sie ein neuer LN-Benutzer sind, ist der Wechsel zu einer anderen neuen LN-Wallet-App manchmal frustrierend, bis Sie sie gelernt haben.

Keine Sorge, es ist nichts falsch daran, von der Bluewallet-App wegzugehen, wenn Sie wirklich den nächsten Schritt auf ein fortgeschritteneres Niveau machen möchten.

Dieser Schritt bedeutet, dass Sie sich für eine Lösung mit mehr Selbstverwahrung entscheiden möchten, und das ist im Bitcoin-Bereich immer willkommen. Mehr Selbstverwahrung bedeutet mehr Souveränität, bedeutet mehr finanzielle Freiheit.

Welche Optionen haben Sie?

  • Mobile nicht verwahrte LN-Knoten: wichtiger Leitfaden hier zu privaten Knoten.
  • Weitere LN-Wallets, wählen Sie mit Bedacht aus diesem Vergleichsleitfaden oder folgen Sie diesem weiteren Leitfaden für Anfänger.

Denken Sie daran: Wählen Sie keine LN-Wallet-App, nur weil Ihnen jemand anderes sie „empfohlen“ hat. Wählen Sie die Wallet-App, die am besten zu Ihrem Anwendungsfall, Ihren Gewohnheiten, Ihrem Ausgabenvolumen über LN usw. passt.

Jeder, der eine Wallet-App „empfiehlt“, kennt Sie oder Ihren speziellen Anwendungsfall für LN nicht oder möchte einfach eine bestimmte App anpreisen.

Verwenden Sie NIEMALS nur eine LN-Wallet-App, verwenden Sie immer mindestens 2-3, um eine Lastverteilung und ein Backup zu haben. Verteilen Sie Ihren LN-Vorrat auf einige davon und seien Sie jederzeit bereit, sie zu verwenden.

D. Aktivieren Sie den versteckten LDK-Knoten in Bluewallet

Ja, Sie haben richtig gehört, Bluewallet hat jetzt einen integrierten LDK-Knoten als Lightning-Wallet. Und das ist gut.

Aber denken Sie daran, es ist noch in der Betaphase, also gehen Sie vorsichtig vor. Wie wird es aktiviert?

Tippen Sie auf das „+“, um Wallet hinzuzufügen, und tippen Sie dann 10 Mal auf die Lightning-Schaltfläche.

Voilà, Sie können dann ein Wallet über LDK erstellen.

Speichern Sie die Seed-Wörter an einem sicheren Ort und jetzt können Sie mit dem Öffnen von Kanälen mit dem Bluewallet LSP-Knoten beginnen. Klicken Sie dazu auf die Schaltfläche „Wallet finanzieren“ und wählen Sie ein beliebiges Onchain-Wallet aus, das Sie bereits in Ihrem Bluewallet konfiguriert haben und das über genügend Guthaben verfügt, um einen LN-Kanal zu öffnen. Sie sollten diese Onchain-Wallet immer mit Ihrem LDK-Knoten verknüpft lassen und sie nicht aus Ihrer Bluewallet-App entfernen. Im Falle der Schließung eines Kanals fließen die Gelder zurück in die Onchain-Wallet, die den Kanal finanziert hat.

Fertig, jetzt können Sie Ihr Bluewallet LDK wie jedes andere reguläre LN-Wallet verwenden, in einem eher selbstverwalteten Modus.
